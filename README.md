水平面に音を自由に配置して、立体的に聴ける空間音響デスクトップアプリ **Sonarena** のテスター版を配布しています。

> ダウンロードは、テスターに個別にご案内したリンクからお願いします。

| OS | ファイル |
|---|---|

| macOS | `macos-build.zip` |

ダウンロードしたら zip を展開し、中の実行ファイルを起動してください。

## 起動方法（重要：警告が出ます）

テスター版は **署名（コードサイニング）を行っていません**。そのため、初回起動時に OS から「提供元が不明」という趣旨の警告が表示されます。これは未署名アプリ共通の挙動で、続行すれば問題なく使えます。

### Windows
1. zip を展開する。
2. 実行ファイルをダブルクリックする。
3. 「Windows によって PC が保護されました」と表示されたら、**「詳細情報」** をクリック →  **「実行」** を押す。

### macOS
1. zip を展開する。
2. アプリを **右クリック（または control + クリック）→「開く」** を選ぶ。
3. 確認ダイアログで再度 **「開く」** を押す。
   - ※ ダブルクリックだと開けない場合があります。必ず右クリックから開いてください。

### Linux
1. zip を展開する。
2. 実行ファイルに実行権限を付与する（例：`chmod +x sonarena`）。
3. 起動する。

## テスターにお願いしたいこと

専門知識は不要です。DAW がなくても試せます。気づいた範囲で、以下のような点を教えていただけると助かります。

- **使い心地**：迷った所・分かりにくい所（例：ボタンの意味が分からない／操作の順番に迷った）
